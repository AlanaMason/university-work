# university-work

В данном репозитории собраны работы по общей теории связи и алгоритмам обработки сигналов.  
Все работы выполнены на MATLAB.

## Алгоритмы обработки сигналов

- Помехоустойчивость приема сигналов с OFDM
- Временная и частотная синхронизация сигналов с OFDM
- Методы снижения пик-фактора OFDM сигналов
- FBMC-OQAM

## Общая теория связи

- Исследование помехоустойчивости и сравнение шенноновской эффективности сигнально-кодовых конструкций
- Приемник сигналов с полным откликом
- Обработка сигналов с CDMA
- Обработка сигналов с OFDM
- Обработка сигналов с MIMO
