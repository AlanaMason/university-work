# Обработка сигналов с MIMO

Структура папки: 
1. model содержит в себе код лабораторной работы на Matlab
2. Results - файл с результатами моделирования. 
3. Файл с описанием задания лабораторной работы

*Что было сделано*
-**Новый класс MIMO, обеспечивающий приемо-предачу SIMO, MISO и MIMO**
-**Для SIMO: метод выбора лучшей антенны, метод суммирования по всем антеннам и метод комбинирования по критерию максимума отношения сигнал-шум**
-**Для MISO: схема Аламоути, метод формирования диаграммы направленности**
-**Для MIMO: ZF, ZF-SIC, ZF-SIC OO, MMSE, MMSE-SIC и MMSE-SIC OO**
-**Имитация многолучевого распространения (EPA/EVA/ETU)**
-**Эквалайзер ZF и MMSE во временной области**

Необходимо было исследовать помехоустойчивость приёма сигналов с SIMO, MISO и MIMO в зависимости от числа антенн, методов обработки и типа детектора. На практике выяснили следующее. 
Для SIMO: увеличение числа приемных антенн увеличивает помехоустойчивость, метод комбинирования по критерию максимума отношения сигнал-шум работает лучше остальных методов, выбор лучшей антенны самый простой, но самый неэффективный. 
Для MISO: схема Аламоути даёт выигрыш в качестве приёма без увеличения числа приёмных антенн. Формирование диаграммы направленности требует знания канала на передатчике. 
Для MIMO: детекторы ZF-SIC, MMSE-SIC работаю лучше, чем ZF и MMSE, особенно при упорядочивании лучей. Если сравнивать конфигурации между собой - MIMO технология самая сложная в обработке, но дает самую большую спектральную эффективность. Также увеличение приеных антенн увеличивает помехоустойчивость, но усложняет обработку.
