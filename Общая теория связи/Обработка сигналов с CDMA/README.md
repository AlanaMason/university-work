# Обработка сигналов с CDMA

Структура папки: 
1. model содержит в себе код лабораторной работы на Matlab
2. Results - файл с результатами моделирования. 
3. Файл с описанием задания лабораторной работы

*Что было сделано*
-**Процедура расширения/дерасширения**
-**Процедура скремблирования/дескремблирования**
-**Имитация многолучевого распространения (EPA/EVA/ETU)**
-**ZF-эквалайзер**
-**Rake-приемник**

Необходимо было исследовать помехоустойчивость приёма CDMA-сигналов в зависимости от коэффициента расширения, типа многолучевого канала, наличия скремблирования и количества лучей в Rake-приёмнике. 
На практике выяснил следкющее. При увеличении коэффициента расширения увеличивается помехоустойчивость. Данные после канала EPA лучше восстанавливаются эквалайзером по сравнению с EVA и ETU, так как разброс задержек меньше. ZF-эквалайзер простой в исполнении,справляется с межсимвольной интерференцией, однако может усиливать шум. Rake-приемник борется с многолучевостью за счет того, что суммирует энергию разных лучей, однако при добавлении уже третьего-четвертого луча, прирост помехоустойчивости становится незначительным.
