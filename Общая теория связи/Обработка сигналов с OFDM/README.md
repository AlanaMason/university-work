# Обработка сигналов с OFDM

Структура папки: 
1. model содержит в себе код лабораторной работы на Matlab
2. Results - файл с результатами моделирования. 
3. Файл с описанием задания лабораторной работы

*Что было сделано*
-**Формирование OFDM символов и обратная ей процедура**
-**Равномерное распределение поднесущих, вставка циклического префикса**
-**ZF и MMSE эквалайзер с выбором метода интерполяции**
-**Сверточное кодирование, LDPC кодирование, перемежение**
-**Многолучевое распространение (EPA/EVA/ETU)**


Необходимо было исследовать помехоустойчивость приёма сигналов с OFDM в зависимости от типа канала, наличия кодирования/перемежения, длины циклического префикса, числа пилотов, типа эквалайзера и доплеровской частоты. На практике выяснили следующее.
Без кодирования в многолучевом канале OFDM подвержен ISI и ICI, вследствие чего падает помехоустойчивость. При добавлении кодирования (особенно LDPC) и перемежения помехоустойчивость увеличилась. Перемежитель эффективен при замираниях, благодаря нему ошибки равномерно разносятся на все блоки. Для циклического префикса существует оптимальная длина для компромисса между помехоустойчивостью и спектральной эффективностью, она зависит от типа канала. ZF эквалайзер проще в исполнении, однако может усиливать шум, MMSE учитывает уровень шума, поэтому на более низких ОСШ работает лучше ZF.
