# Исследование помехоустойчивости и сравнение шенноновской эффективности сигнально-кодовых конструкций

Структура папки: 
1. model содержит в себе код лабораторной работы на Matlab
2. Results - файл с результатами моделирования. 
3. Файл с описанием задания лабораторной работы

*Что было сделано*
-**Кодер/декодер сверточных кодов (полином [171, 133])**
-**LDPC кодер/декодер (стандарт DVB-S2)**
-**Модулятор/демодулятор созвездий PSK и QAM, поддержка разных вариантов решений (мягкие, жесткие)**
-**Перемежитель**

Необходимо было исследовать помехоустойчивость приема сигналов для созвездий ФМ-4 и КАМ-16, и как на нее влияют различные конфигурации, такие как кодирование, тип решений демодулятора. 
На практике выяснили, что мягкие решения демодулятора дают более точный результат по сравнению с жесткми, однако требуют знания информации о канале. LDPC кодирование надежнее сверточного, однако работает медленнее и сложнее в реализации. Также перемежитель в канале с АБГШ не дал прироста в помехоустойчивости, что логично.
