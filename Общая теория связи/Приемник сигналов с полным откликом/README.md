# Приемник сигналов с полным откликом

Структура папки: 
1. model содержит в себе код лабораторной работы на Matlab
2. Results - файл с результатами моделирования. 
3. Файл с описанием задания лабораторной работы

*Что было сделано*
-**RRC-фильтр**
-**Неидеальная синхронизация по частоте**
-**Неидеальная синхронизация по времени**

Необходимо было исследовать помехоустойчивость приёма сигналов с полным откликом в зависимости от параметров RRC-фильтра и точности временной и частотной синхронизации.
На практике выяснили следующее. Коэффициент сглаживания (beta) влияет на ширину полосы пропускания фильтра и уровень боковых лепестков. Чем меньше beta, тем уже полоса, но тем более точная синхронизация требуется. Из-за рассогласования по частоте происходит поворот созвездия, вызывающий рост количества ошибок. Некогерентные схемы модуляции более устойчивы к рассогласованиям по частоте.
