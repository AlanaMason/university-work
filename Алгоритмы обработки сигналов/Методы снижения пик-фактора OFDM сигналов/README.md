# Методы снижения пик-фактора OFDM сигналов

Структура папки: 
1. Matlab файлы с кодом лабы.
2. Results - файл с результатами моделирования. 
3. Файл с описанием задания лабораторной работы

*Что было сделано*
-**Расчёт CCDF пик-фактора**
-**Расчёт спектральной плотности мощности**
-**Метод клиппирования с последующей фильтрацией**
-**Метод SLM (Selected mapping)**
-**Метод PTS (Partial Transmit Sequences)**

Клиппирование самый простой метод, который обрезает пики выше заданного порога. При слишком сильном обрезании возникают внеполосные излучения из-за нелинейности процедуры. Многоитерационное клиппирование с фильтрацией уменьшает внеполосные излучения, но увеличивает вычислительную сложность.
SLM генерирует несколько вариантов одного OFDM символа с разными фазовыми поворотами (фазы {0, π}) и выбирает вариант с минимальным PAPR. Чем больше последовательностей, тем сильнее снижение PAPR. PTS разбивает поднесущие на блоки, каждый блок умножается на свой фазовый множитель (фазы {0, π}), затем выбирается лучшая комбинация. Чем больше блоков, тем сильнее снижение PAPR. Минус этих подходов: на приеме нужно знать формирующие последовательности, чтобы восстановить сигнал обратно.
