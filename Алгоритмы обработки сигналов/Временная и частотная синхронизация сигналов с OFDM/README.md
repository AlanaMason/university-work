# Временная и частотная синхронизация сигналов с OFDM в каналах с АБГШ

Структура папки: 
1. Matlab файлы с кодом лабы.
2. Results - файл с результатами моделирования. 
3. Файл с описанием задания лабораторной работы

*Что было сделано*
-**Формирование OFDM символов, равномерное распределение пилотных поднесущих, добавление циклического префикса**
-**Многолучевое распространение (EPA/EVA/ETU)**
-**Алгоритм временной синхронизации по циклическому префиксу**
-**Алгоритм временной синхронизации Минна**
-**Алгоритм частотной синхронизации по циклическому префиксу**
-**Алгоритм частотной синхронизации на основе периодических повторений**

Часть 1: Временная синхронизация.
При временной рассинхронизации происходит смещение окна БПФ, что приводит к ISI и ухудшению точности приема. Для алгоритма синхронизации по циклическому префиксу автокорреляционная функция имеет пик на расстоянии, равном длине полезной части символа. Чем больше символов используется для синхронизации, тем точнее оценка, но тем медленнее работа алгоритма. Для алгоритма Минна автокорреляционная функция даёт более острый пик, чем метод на CP, за счёт использования специальной структуры символа с повторениями. Алгоритм хорошо работает даже при низком отношении сигнал/шум. 

Часть 2: Частотная синхронизация
При частотной рассинхронизации разрушается ортогональность между поднесущими, что приводит к ICI. Алгоритм по циклическому префиксу работает в диапазоне нормированной расстройки [-0.5, 0.5]. Чем больше символов используется и чем длиннее CP, тем точнее оценка. Алгоритм на основе периодических повторений способен оценивать больший диапазон расстроек. Точность зависит от структуры символа. 

Для надежной работы OFDM требует как временную, так и частотную синхронизацию.
