# Помехоустойчивость приема сигналов с OFDM в каналах с АБГШ и в каналах с замираниями

Структура папки: 
1. Matlab файлы с кодом лабы.
2. Results - файл с результатами моделирования. 
3. Файл с описанием задания лабораторной работы

*Что было сделано*
-**OFDM модулятор/демодулятор**
-**Многолучевые каналы (EPA/EVA/ETU)**
-**Оценка канала по пилотам**

Без защитного интервала в каналах с многолучевым распространением возникает ISI. Циклический префикс сохраняет ортогональность между поднесущими, длину следует выбирать в соответствии с каналом распространения, чтобы он не оказался слишком коротким. А слишком длинный циклический префикс спектрально неэффективен. При увеличении доплеровского сдвига канал начинает быстрее меняться, и чтобы оценка по пилотным поднесущим продолжала работать, их нужно вставлять чаще.
